A professional, multi-model Machine Learning healthcare system built with **Streamlit**, **Scikit-Learn**, and **Plotly**. This application evaluates patient profiles and predicts the risk of diabetes using advanced ensemble and classification algorithms.

---

## ✨ Features

- **Multi-Model Selection**: Choose between **Random Forest**, **Logistic Regression**, and **K-Nearest Neighbors (KNN)** to analyze predictive metrics dynamically.
- **Interactive UI**: A stunning, modern, and responsive user interface featuring glassmorphism and premium styling.
- **Real-Time Analytics**: Visualizes dataset distributions and model decisions through highly interactive Plotly graphs.
- **Dynamic Feature Importance**: Explains predictions by mapping out feature weights and decision bounds (for Random Forest and Logistic Regression).
- **Clinical Suggestions**: Provides automated, medically conscious feedback based on the predicted risk probability.

## 🚀 Quick Start (Local Development)

### 1. Clone the repository
```bash
git clone https://github.com/yourusername/Diabetes_Prediction_Project.git
cd Diabetes_Prediction_Project
```

### 2. Install dependencies
Ensure you have Python installed. Then, run the following to install the required libraries:
```bash
pip install -r requirements.txt
```

### 3. Run the application
Start the Streamlit server:
```bash
streamlit run app.py
```
*The app will automatically open in your default web browser at `http://localhost:8501`.*

## 🛠️ Tech Stack

- **Frontend / Framework**: [Streamlit](https://streamlit.io/)
- **Machine Learning**: [Scikit-Learn](https://scikit-learn.org/), [Pandas](https://pandas.pydata.org/), [NumPy](https://numpy.org/)
- **Data Visualization**: [Plotly](https://plotly.com/python/)
- **Styling**: Vanilla CSS injected natively via Streamlit Markdown
